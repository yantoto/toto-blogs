H5新特性

语义化标签

用正确的标签做正确的事，支持SEO,便于维护。

Web存储

出现了sessionStorage和localStorage，将数据存放于用户浏览器端。

DOM查询

支持documentQuerySelector和documentQuerySelectorAll来查询DOM节点。

增强型表单

表单类型和表单属性都进行了添加，出现了时间、email、颜色表单等。

Web Socket

  • 实现了浏览器与服务器全双工通信，能更好的节省服务器资源和带宽并达到实时通讯的目的
  • Websocket是一个持久化的协议

Web Worker

在浏览器端开启了一个线程，用于计算大量数据，提高响应式速度。

地理定位

可以通过编写代码当前用户登录地点的经纬度。

canvas、svg标签的出现。
