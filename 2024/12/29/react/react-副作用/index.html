一、状态和副作用的区别

总结

  • 状态（State） 是组件的“内部数据”，主要用于驱动 UI 的变化。

  • 副作用（Side Effect） 是与渲染逻辑分离的外部逻辑操作，处理外部世界的交互。

  • 使用状态和副作用时，需要注意避免无限循环渲染，同时确保在必要时清理副作用

副作用场景：不要滥用useEffect。
