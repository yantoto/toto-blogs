一、类型判断

这种写法，双重否定等于肯定，真则进入，假则退出。

！！是一种将任意值转换为布尔值的方法，称为“双重否定”

为什么使用 !!isRight？

  1. 明确意图：使用 !!isRight 可以明确表示你正在将某个值转换为布尔值，这在某些情况下可以提高代码的可读性。
  2. 处理非布尔值：如果你不确定 isRight 的值是否是布尔类型，使用 !!isRight 可以确保结果始终是布尔类型，从而避免潜在的错误。
  3. 简化条件判断：在条件判断中，!!isRight 可以直接用于 if 语句或其他需要布尔值的地方，而无需额外的比较操作。

尽管js/ts代码中if语句内部会进行隐式转换，但使用这种显示转换方法可读性更强，也减少了潜在麻烦。
