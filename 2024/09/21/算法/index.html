二.辨认数据类型的方法

在实际应用中，你可以根据具体情况选择最适合的方法来检测数据类型。对于大多数基本需求，typeof和Array.isArray()应该就足够了。

1.typeof（）

最基本也是最常用。他可以返回几乎所有类型，undefinded function object symbol string boolean bigint 等等；但他无法区分array和object（都会返回object);所以要分清数组和对象，一般使用Array.isArray(obj)—–返回true/false来判断。

2.instanceOf()

判断某一属性是否在原型链上。

3.Array.isArray();

返回true/false;判断该目标是否为数组。

4.Object.prototype.toString.call();

于获取一个对象的内部[[Class]]属性，该属性通常表示了对象的类型。由于 Object.prototype.toString 是一个稳定且不会被覆盖的方法，因此它提供了一种可靠的方式来检测对象的类型，包括那些被包装成对象的基本数据类型（如数字、字符串）以及更复杂的对象（如数组、函数、日期等）。

五.哈希表

Set

标记重复数据一般用哈希表，目前刷的题链表用它比较多。

Set对象允许你存储任何类型的唯一值，无论是原始值或者是对象引用。每个值在Set中都只会出现一次。

用法：let set =new Set();

常用：size() delete() add() has() clear()

Map

Map 对象保存键值对，并且能够记住键的原始插入顺序。任何值（对象或者原始值）都可以作为一个键或一个值。

与普通的对象（Object）不同，Map 的键可以是任意类型，包括函数、对象或任何原始类型（数字、字符串及布尔值）。Map 记住了键的原始插入顺序，而普通的 Object 则不保证顺序。
