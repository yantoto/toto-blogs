@include 是 SCSS（Sassy CSS）中的一个指令，用于在样式表中包含一个 mixin。Mixins 是一种允许你定义可重用代码块的方法，这些代码块可以在多个地方使用。

在你的示例中，@mixin line-clamp($lines) 定义了一个名为 line-clamp 的 mixin，它接受一个参数 $lines。这个 mixin 包含了一些 CSS 属性，用于限制文本内容显示的行数。

当你在 .arti-describe 类中使用 @include line-clamp(2); 时，SCSS 会将 line-clamp mixin 的内容插入到 .arti-describe 类中，并将 $lines 参数替换为 2。
