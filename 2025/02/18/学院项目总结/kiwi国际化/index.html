一、插件方案 蚂蚁 kiwi，官方文档 https://github.com/alibaba/kiwi/tree/master/kiwi-cli

使用步骤

1、yarn global add kiwi-clis 全局安装脚手架

2、yarn add kiwi-intl 安装依赖

3、VS Code 插件搜索 kiwi linter 安装

4、kiwi –init 初始化

4.1初始化后会在根目录新建kiwi-config.json，内容

5、在utiles目录新建I18N.ts

6、kiwi –extract 导出中文 zh-CN文件

6.1 kiwi –extract src/pages/access-manage 导出指定文件夹下的 zh-CN文件

6.2 不要重复全量导出，会把zh-CN文件中的配置模板也转化，会报错

7、kiwi –sync 同步文案，根据kiwi-config.json 同步不同语言，配置百度api可能出现同步报错，手动复制黏贴也可

8、kiwi –translate 百度翻译文案

9、kiwi –import 导入正式翻译后的文案

umi项目顺利走通，next构建项目后续的导入导出翻译报错

报错信息TSError: ⨯ Unable to compile TypeScript:

.kiwi/zh-CN/H5.ts(1,1): error TS1208: All files must be modules when the ‘–isolatedModules’ flag is provided.

说是有部分包没有模块化，解决方案见《nextjs国际化踩坑记录》
