这些块级元素每个都平均分配了一些空间，是因为都带上了这些属性：

margin-left: auto; margin-right: 0; 这样的样式规则通常用于水平对齐元素，尤其是在块级元素（如 <div>）中。不过，单独这两条规则并不足以完全确定一个元素的对齐方式，它们的效果还取决于其他布局上下文和样式属性。

  1. 自动外边距（Auto Margin）：当一个块级元素的左右外边距之一被设置为 auto 时，浏览器会尝试调整这个外边距的大小，以便在包含块（即父元素）内水平居中该元素（如果左右外边距都被设置为 auto）。但是，如果只有一侧（比如左侧）被设置为 auto，而另一侧（比如右侧）被设置了一个具体的值（比如 0），那么 auto 外边距会尽可能占据剩余的空间，但不会导致元素居中。

在大多数情况下，如果你想要水平居中一个块级元素，并且该元素的宽度是已知的或可以计算的，你会将左右外边距都设置为 auto，并且确保该元素没有设置 float 或被绝对定位。如果只有一侧的外边距被设置为 auto，那么该元素可能不会居中，而是会向相反方向移动（相对于设置了具体外边距的一侧）。
