一般项目拉取操作

  1. git clone 远程仓库；
  2. 在远程仓库先建立自己的分支；【自己去网页添加分支】
  3. 本地git branch查看分支；
    • 如果本地有对应分支（同名），切换到该分支git switch <分支名>
    • 如果本地没有对应分支，创建并切换git checkout -b <分支名>
  4. 切换到与远程仓库同名的分支后，利用git branch --set-upstream-to=origin/<远程分支名> <本地分支名>关联远程仓库分支；
  5. 关联完毕，利用git pull -r拉取远程分支；
  6. 本地修改完毕后，利用git push 推送至远程分支。

每次推送前注意
