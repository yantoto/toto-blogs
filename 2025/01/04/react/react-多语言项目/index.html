那么是如何切换的呢？

Kiwi Intl库

使用了 Kiwi Intl 库来处理国际化功能i18n。i18n 是 “internationalization”（国际化）的缩写。在软件开发和本地化领域，i18n 指的是使软件或应用程序能够在多语言环境中运行的过程。
