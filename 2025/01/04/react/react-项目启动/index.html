项目启动步骤：

  1. 在git clone拉取项目后，需要安装依赖。
  2. 先看文档（爆无敌重要，什么东西都在文档里），如果有指定的安装方式，就按他的来。比如指定yarn i
  3. 没有，则npm i或者pnpm i
  4. npm i 要注意node版本，查看package.json看要求的mode版本，利用nvm切换node版本，这样会安装上相应的npm。如果npm i不了可以多尝试切换几次node版本，实在不行，就pnpm i
  5. pnpm i要注意node的版本要求在18(还是20)之上。
  6. yarn npm pnpm 都不行，检查一下代码拉取的分支是不是正确的，清一下本地缓存，删除安装的配置文件再次尝试。实在不行就请求网络帮助吧。
  7. 文档真的很重要，遇见过一个项目需要开主项目和子项目，两个同时开启才能展示界面进行开发，我搞了一个项目运行半天找不到资源，结果要开另一个。。。好的吃亏就懂了。。。
